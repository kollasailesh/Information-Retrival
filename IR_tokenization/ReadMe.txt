Read Me

1.	There are two class files named Tokenizer.java and stemmer.java. Please copy them into your directory. 
2.	Open command prompt and navigate to the directory having these two files. 
3.	First compile Stemmer.java file with the command 
	>javac Stemmer.java 
4.	Next, compile Tokenizer.java file with the command 
	>javac Tokenizer.java
5.	Next, run the TokenizerClass class.
	>java Tokenizer 		 

If the location of the cranfield documents is different, please specify it in the Tokenizer class.
